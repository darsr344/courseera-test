Google Chrome Troubleshooting

An issue has been reported where Google Chrome is not working.

Check if user has a business justification to use Google Chrome

  1. If YES then go ahead and follow the below mentioned steps
  2. else, inform the user IE is the standard browser for WG
  3. If User has business justification for using chrome then go ahead and grant local admin rights to the user
  4. Log off and log in back
  5. Launch EMET
  6. Click on "APPS" option on the top ribbon
  7. Uncheck all the options against chrome
  8. Relaunch Chrome
  9. It should work fine
  10. Once this has been done, remove the user from local admin rights

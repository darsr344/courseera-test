Outlook Troubleshooting

Files and Folders Missing

  1. Remote into User's computer
  2. Click on Start and then select Run
  3. Type "Outlook.exe /cleanviews"
  4. Before performing any of the above steps close outlook and then proceed
  5. Once the above switch command is keyed ou would relaunch
  6. User should now be able to see the missing emails and folders
  7. The abpve mentioned Outlook Switch resolved any issues related to viewing an email or folder in Outlook
